
Recommender Utilities
===================================================

The `rater repository <https://github.com/shibing624/rater>`_ provides examples and best practices for building recommendation systems, provided as Jupyter notebooks.

The module `recommender <https://github.com/shibing624/rater/tree/master/rater>`_ contains functions to simplify common tasks used when developing and
evaluating recommender systems.
